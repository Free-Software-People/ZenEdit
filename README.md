# ZenEdit

A simple text editor that has all the features you need without being in your face.

## Features
- Minimalist, distraction-free interface.
- No AI, no bright colors.
- Lightweight C++ implementation using Qt.
- Open, Save, and Save As functionality.

## Build Instructions
ZenEdit uses the Meson build system and Ninja.

```bash
meson setup build
ninja -C build
./build/zenedit
```
