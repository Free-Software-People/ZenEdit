# ZenEdit User Guide

ZenEdit is designed to be a minimalist text editor.

## Basic Usage
- **Open a File**: Go to `File` -> `Open` or press `Ctrl+O`.
- **Save a File**: Go to `File` -> `Save` or press `Ctrl+S`.
- **Save As**: Go to `File` -> `Save As...` to save the current content to a new file.
- **Revert**: Go to `File` -> `Revert` to reload the file from disk, discarding unsaved changes.
- **Undo**: Go to `Edit` -> `Undo` or press `Ctrl+Z`.
- **Redo**: Go to `Edit` -> `Redo` or press `Ctrl+Y`.
- **Exit**: Go to `File` -> `Exit`.

## Philosophy
ZenCode avoids all unnecessary clutter. There are no complex toolbars, no AI assistants, and no jarring colors—just you and your text.
